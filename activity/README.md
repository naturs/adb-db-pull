# Adb-Activity #

## 描述 ##

`Adb-Activity`主要是用来处理`Activity`相关的命令。

## 安装 ##

	sudo cp ./adb-current-activity /usr/local/bin/

	sudo chmod +x /usr/local/bin/adb-current-activity

## 使用 ##

轮询获取当前`Activity`：

	adb-current-activity
